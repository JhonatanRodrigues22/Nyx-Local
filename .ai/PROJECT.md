# Project

## Name

Nyx Local

## Summary

Nyx Local is a local-first Python application built through small Sprints and maintained through an AI-friendly documentation system.

## Current Phase

Local communication foundation.

## Current Capabilities

- Layered Python project foundation.
- Application bootstrap infrastructure.
- Request and Response flow.
- Console response rendering.
- Registry and Settings.
- JSON-backed persistent memory foundation.
- Intelligence Pipeline foundation.
- Nyx OS local gateway WebSocket client.
- Minimal Skill Runtime with `local.echo`.
- Git/Pull Request preparation workflow.
- AI collaboration documentation.

## Current Restrictions

Do not implement new application behavior during documentation-only Meta Sprints.

Operating-system automation, LLM integration, additional providers, vector memory, semantic search, Obsidian integration, and advanced memory workflows require explicit Sprint approval.
